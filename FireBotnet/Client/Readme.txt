ENGLISH:

This ocx control is designed to make skins in Visual Basic 6 forms in a very simple way.
Copy the file vbskfr2.ocx into the Windows\System32 folder and vbskfr2.chm into the Windows\Help folder, or place them in the folder that you want.
To include the control in a VB project go to the 'Project' menu, select 'Components', then press browse and select the file vbskfr2.ocx. After that you will be able to add the control to the forms.


The files included in the package:

1) this file Readme.txt
2) vbskfr2.ocx (program file)
3) vbskfr2.chm (help file)
4) vbskfr2.DEP (dependency file)

ESPAÑOL:

Este control ocx está diseñado para hacer skins en formularios de Visual Basic 6 de una manera muy sencilla.
Copie el archivo vbskfr2.ocx en la carpeta Windows\System32 y el archivo vbskfr2.chm en Windows\Help, o déjelos en la carpeta que prefiera.
Para incluir el control en un proyecto VB diríjase al menú 'Proyecto', seleccione 'Componentes', luego presione el botón 'Examinar', busque y selecccione vbskfr2.ocx. Luego de eso estará en condiciones de agregar el control a los formularios.

Los archivos incluidos en este paquete son:

1) este archivo Readme.txt
2) vbskfr2.ocx (archivo del programa)
3) vbskfr2.chm (archivo de ayuda)
4) vbskfr2.DEP (archivo de dependencias)

Sitio web: www.visual-basic.com.ar
Autor: Javier Balkenende
